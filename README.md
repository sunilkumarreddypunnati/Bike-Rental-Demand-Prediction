# Bike Rental Demand Prediction

This project analyses the Capital Bikeshare dataset to understand rental patterns and predict daily bike rental demand using machine learning.

## Dataset
Source: UCI Machine Learning Repository

Files used:
- day.csv (731 daily records)
- hour.csv (17,379 hourly records)

## Tools & Libraries
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Project Workflow
1. Data loading and preprocessing
2. Exploratory Data Analysis (EDA)
3. Feature selection
4. Model building
5. Model evaluation

## Machine Learning Models
- Linear Regression
- Random Forest

## Results
Random Forest performed better with:

R² Score: **0.885**

## Key Insights
- Bike rentals increase during summer and fall
- Weather conditions strongly affect rental demand
- Registered users dominate rental usage
